# 📷 Camera Landing Page

A simple and responsive Camera Landing Page created while learning HTML and CSS. This beginner-friendly project focuses on building a clean user interface using only HTML and CSS, without any JavaScript.

## Features

* Responsive layout
* Modern landing page design
* Clean and organized structure
* Navigation bar
* Hero section with camera showcase
* CSS styling with hover effects

## Technologies Used

* HTML5
* CSS3

## Purpose

This project was built to practice HTML and CSS fundamentals, including page structure, layouts, styling, spacing, typography, and responsive design concepts.
